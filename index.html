HTML Basics Lesson 1

The Internet

The World Wide Web is the largest computer network in the world. The web works as a series of four component parts.
You, your computer/browser, the Internet itself, and the servers all make up what is collectively known as the Internet.
Your browser is a program that runs on your computer to display files found on the web. Servers are computers that
host these files. Your browser makes a request to these servers using HTTP, the main protocol of the web.
This video explains this well.

HTML

HTML is an acronym for "Hypertext markup language." HTML code is the basis for almost every page on the web.
HTML consists of the text content of the page, the markup for that text, references to other documents, and links to
other pages on the web. HTML is coded using elements. HTML tags are a type of element that defines the format/display
of the content of that page.

Inline and Block Elements

HTML elements can be either inline of block. Block elements create an invisible box around a body of text. The break
tag is an inline element, while the paragraph tag is a block element.

Note on Coding

Because computers do not think like we do, tiny mistakes in code, like not forgetting a closing tag, become big mistakes
for computers. Although a mistake may seem insignificant to us, it can yield an entire section of code meaningless to a
computer. Secondly, it is important to remember that eliminating repitition is fundamental to good coding. Reptition in your
HTML, CSS, or both can be detrimental to your webpage.

Notes for Creating a Structured Document Lesson 2

Thinking of a Website as a House

A house and a website may not seem very similar, but in reality, a house is a great analogy for a website. Think of the entire
house as the website itself. Now consider the two programming languages - CSS and HTML. If the website is the whole house
then the HTML is the structure - walls, rooms, etc. - and the CSS is the style of the home - color, decorations, etc.

Similarly, HTML is the structure of the webpage, while CSS is the formatting and style. This video explains this concept.

Analyzing Pages on the Web

To analyze the HTML structure of any webpage online, you can right click on the page and go to the "inspect element"option.
A new window will open on the page containing all the HTML code, as well as the stylings (CSS). If you click the triangles to the
left of the code and continue to do so for all of the triangles on the page, the DOM (Document Object Model) structure of the page
will appear. This is the tree like structure of the webpage.

Looking at a Webpage in terms of Boxes

Everything on a webpage is rectangular. Even circles and other shapes seen on a webpage are contained in boxes. We define
the boxes of a webpage using the "div" tag.

Adding CSS Style to your Code Lesson 3

CSS Basics

CSS stands for Cascading Style Sheets. CSS is a programming language that adds style to the structure of an HTML document.
A link to your CSS code should be put in the header section of HTML code. CSS properties can be added to HTML using selecotrs,
which are keywords that apply a CSS property to all that apply on that page. The CSS selectors can be a tag or a class. If the selector is
a tag, refer to it without a period before it. If the selector is a class it should be referred to with a period before it. The CSS properties
should be referred to in the format: "attribute.value."

What to do with CSS

CSS can be used to change text color, font, border radius, the margins of your page, the font weight, the text alignment, and much more.
In order to manipulate these properties, you must select the correct attribute and assign it a value.

The Box Model

Because every HTML document can be broken down into a series of boxes, and CSS can be used to manipulate these boxes, it is important
to define the parts of boxes found on the internet. All HTML boxes are composed of the content, padding, border, and margin. The content is
the text and imaging that appears within the box. The padding clears an area around the contnet. The border clears an area around
the padding, and the margin is the clear space between the boxes themselves.
